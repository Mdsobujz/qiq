# qiq
#### Package installer instead of:  pkg,  apt,  apt-get
---
## Installing for termux
* `pkg install git`
* `git clone https://github.com/mishakorzik/qiq`
* `cd qiq`
* `bash install.sh`

* `Succesfull installed!`

----
## Installing for linux
* `apt install git`
* `git clone https://github.com/mishakorzik/qiq`
* `cd qiq`
* `bash install.sh`

* `Succesfull installed!`

-----
